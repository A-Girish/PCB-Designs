* Manufacturable board design

This repository serves as:

## 📐 Design Approach

While designing these PCBs, I try to follow common **industry practices** such as:

* Separating functional circuit blocks in the schematic
* Proper grounding and return paths
* Decoupling and bypass capacitors near ICs
* Clear signal routing
* Adequate trace width for current requirements
* Running design rule checks before manufacturing

Some early designs may not fully follow best practices as they represent part of the **learning process**.
